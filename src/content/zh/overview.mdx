# 概览

Claude Code 是 Anthropic 开发的智能编程工具，运行在您的终端中。它能够理解您的代码库、读写文件、运行测试，并通过自然语言完成复杂的多步骤任务。

## Claude Code 能做什么

- **读写文件** —— 操作整个项目
- **执行 Shell 命令** —— 测试、构建、代码检查、Git 操作
- **搜索和导航** —— 精准定位大型代码库
- **启动子智能体** —— 并行执行多步骤工作流
- **连接 MCP 服务器** —— 扩展能力（数据库、API、浏览器等）

## 架构

Claude Code 作为 CLI 进程运行在本地机器上：

1. 读取项目上下文（文件、Git 历史、CLAUDE.md 记忆）
2. 向 Anthropic API 发送请求（Claude 模型）
3. 在本地执行工具（文件读写、Bash、搜索）
4. 将结果流式传输到终端

```
您 ──► claude CLI ──► Anthropic API（Claude 模型）
                            │
                       工具执行
                      （本地进程）
                            │
                    文件系统 / Shell
```

## 核心概念

| 概念 | 说明 |
|---|---|
| **会话** | 每次对话为一个会话；使用 `--continue` 恢复上次会话 |
| **CLAUDE.md** | Claude 读取的 Markdown 文件，用于持久化项目上下文 |
| **钩子** | 在工具调用前后运行的 Shell 脚本 |
| **MCP** | 模型上下文协议 —— 插件式工具服务器 |
| **斜杠命令** | 会话内置的 `/命令` |
| **无头模式** | 非交互式的 `claude -p`，适用于 CI/脚本 |

## 模型

Claude Code 默认使用最新可用的 Claude 模型。截至 2025 年：

- **claude-opus-4-6** —— 最强大，适合复杂智能体任务
- **claude-sonnet-4-6** —— 速度与能力均衡（多数场景的默认选择）
- **claude-haiku-4-5** —— 快速、高性价比，适合简单任务

<Callout type="info">
  Claude Code 使用 `ANTHROPIC_API_KEY` 环境变量进行身份验证。请在 Shell 配置文件或 `.env` 文件中设置该变量。
</Callout>

## 权限模型

Claude Code 以您的用户权限运行。在执行以下操作前会请求确认：

- 删除文件
- 运行破坏性 Shell 命令
- 向 Anthropic API 以外的地址发起网络请求

您可以在 `settings.json` 中调整权限行为。
