# 配置

Claude Code 使用分层配置系统，按以下顺序合并（后者优先级更高）：

1. **系统默认值**
2. **用户级** `~/.claude/settings.json`
3. **项目级** `.claude/settings.json`
4. **CLI 参数**（最高优先级）

## 配置文件位置

| 作用域 | 路径 |
|---|---|
| 用户（全局） | `~/.claude/settings.json` |
| 项目 | `.claude/settings.json`（提交到 Git） |
| 项目（本地） | `.claude/settings.local.json`（加入 .gitignore） |

## 核心配置项

```json
{
  "model": "claude-sonnet-4-6",
  "theme": "dark",
  "stream": true,
  "verbose": false,

  "permissions": {
    "allow": ["Read", "Glob", "Grep", "Write", "Edit", "Bash"],
    "deny": []
  },

  "env": {
    "NODE_ENV": "development"
  }
}
```

## 权限配置

控制 Claude 可以无需确认使用的工具：

```json
{
  "permissions": {
    "allow": [
      "Read",
      "Write",
      "Edit",
      "Bash(npm test)",
      "Bash(npm run lint)"
    ],
    "deny": [
      "Bash(rm -rf*)",
      "WebFetch"
    ]
  }
}
```

工具规则支持 Glob 风格的模式匹配。`Bash(npm*)` 允许所有以 `npm` 开头的命令。

<Callout type="warning">
  `Bash` 权限会执行真实的 Shell 命令，请谨慎使用。建议在项目级配置中限制到具体命令。
</Callout>

## 自定义斜杠命令

```json
{
  "customCommands": [
    {
      "name": "review",
      "description": "审查暂存的更改",
      "prompt": "审查我暂存的 Git 更改，查找 Bug、安全问题和代码风格问题。"
    }
  ]
}
```

## 钩子配置

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Write|Edit",
        "hooks": [
          {
            "type": "command",
            "command": "npm run lint --fix 2>/dev/null || true"
          }
        ]
      }
    ]
  }
}
```

详情请参阅[钩子](/zh/hooks)页面。

## MCP 服务器配置

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/tmp"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_..."
      }
    }
  }
}
```

## 环境变量

| 变量 | 说明 |
|---|---|
| `ANTHROPIC_API_KEY` | 必填 —— 您的 Anthropic API 密钥 |
| `ANTHROPIC_BASE_URL` | 覆盖 API 基础 URL |
| `ANTHROPIC_MODEL` | 默认使用的模型 |
| `CLAUDE_CONFIG_DIR` | 覆盖配置目录（默认：`~/.claude`） |
| `NO_COLOR` | 禁用彩色输出 |
| `CLAUDE_DEBUG` | 启用调试日志 |

## 查看当前配置

```
/config
```

或通过 Shell 查看：

```bash
cat ~/.claude/settings.json
cat .claude/settings.json
```
