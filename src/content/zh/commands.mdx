# 命令与斜杠命令

## CLI 参数

运行 `claude --help` 查看所有参数。最常用的参数：

| 参数 | 说明 |
|---|---|
| `--model <id>` | 选择特定的 Claude 模型 |
| `-p, --print <提示词>` | 非交互式（无头）模式 |
| `--continue` | 恢复最近的会话 |
| `--resume <id>` | 按 ID 恢复指定会话 |
| `--no-stream` | 禁用流式输出 |
| `--output-format <fmt>` | 输出格式：`text`、`json`、`stream-json` |
| `--max-turns <n>` | 无头模式下限制智能体轮次 |
| `--allowedTools <列表>` | 白名单：仅允许指定工具 |
| `--disallowedTools <列表>` | 黑名单：禁止指定工具 |
| `--verbose` | 显示完整工具调用详情 |
| `--dangerously-skip-permissions` | 跳过所有确认提示（仅限 CI 使用） |
| `--add-dir <路径>` | 添加额外目录到上下文 |
| `--version` | 显示版本号 |

## 斜杠命令（会话内）

这些命令在活跃的 Claude Code 会话中输入使用。

### 导航与帮助

| 命令 | 说明 |
|---|---|
| `/help` | 列出所有斜杠命令 |
| `/status` | 显示当前会话信息、模型、费用 |
| `/history` | 显示对话历史 |

### 记忆

| 命令 | 说明 |
|---|---|
| `/memory` | 查看和编辑 CLAUDE.md 记忆文件 |
| `/memory add <内容>` | 向最近的 CLAUDE.md 追加备注 |

### 配置

| 命令 | 说明 |
|---|---|
| `/config` | 查看当前配置 |
| `/model <id>` | 会话中途切换模型 |
| `/permissions` | 查看或调整工具权限 |

### MCP

| 命令 | 说明 |
|---|---|
| `/mcp` | 列出已连接的 MCP 服务器和工具 |

### 会话

| 命令 | 说明 |
|---|---|
| `/clear` | 清除对话历史（保留上下文） |
| `/reset` | 重置对话和上下文 |
| `/compact` | 压缩历史以节省上下文窗口 |
| `/cost` | 显示本次会话的 API 费用 |

### 代码与工具

| 命令 | 说明 |
|---|---|
| `/review` | 让 Claude 审查最近的变更 |
| `/diff` | 显示当前 Git diff |
| `/commit` | 暂存并提交更改，自动生成提交信息 |

### 自定义斜杠命令

在 `settings.json` 中定义自己的斜杠命令：

```json
{
  "customCommands": [
    {
      "name": "test-and-fix",
      "description": "运行测试并修复失败",
      "prompt": "运行所有测试。如果有失败，诊断并修复它们。然后再次运行以确认通过。"
    }
  ]
}
```

使用 `/test-and-fix` 调用。

<Callout type="tip">
  自定义命令在配合 IDE 扩展使用时，可包含 `$SELECTION`（编辑器选中内容）和 `$FILE`（当前文件）占位符。
</Callout>

## 工具名称（用于 `--allowedTools`）

限制工具时，使用以下名称：

| 工具 | 功能 |
|---|---|
| `Read` | 读取文件内容 |
| `Write` | 写入/覆盖文件 |
| `Edit` | 对文件进行精准编辑 |
| `Bash` | 执行 Shell 命令 |
| `Glob` | 按模式查找文件 |
| `Grep` | 搜索文件内容 |
| `WebFetch` | 获取 URL 内容 |
| `WebSearch` | 搜索网络 |
| `Agent` | 启动子智能体 |

```bash
# 仅允许读取和搜索工具（安全只读模式）
claude --allowedTools "Read,Glob,Grep"
```
