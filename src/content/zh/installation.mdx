# 安装

## 环境要求

- **Node.js** 18 或更高版本
- **npm** 8 或更高版本
- **Anthropic API 密钥**（在 console.anthropic.com 获取）

## 安装

```bash
npm install -g @anthropic-ai/claude-code
```

验证安装：

```bash
claude --version
```

## 身份验证

将 API 密钥设置为环境变量：

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

添加到 Shell 配置文件以持久保存：

```bash
# ~/.zshrc 或 ~/.bashrc
export ANTHROPIC_API_KEY="sk-ant-..."
```

<Callout type="tip">
  在 macOS 上，您也可以将密钥存储在系统钥匙串中，并通过 `$(security find-generic-password -s anthropic -w)` 引用。
</Callout>

## 首次运行

```bash
cd 您的项目目录
claude
```

Claude Code 会显示交互式提示符。输入任务并按回车键即可开始。

## 更新

```bash
npm update -g @anthropic-ai/claude-code
```

## 卸载

```bash
npm uninstall -g @anthropic-ai/claude-code
```

## IDE 集成

Claude Code 可与以下工具集成：

- **VS Code** —— 从应用市场安装 Claude Code 扩展
- **JetBrains IDE** —— 安装 Claude Code 插件
- **任意终端** —— CLI 可独立使用

<Callout type="info">
  VS Code 和 JetBrains 扩展允许 Claude Code 读取您的打开文件和编辑器状态，作为额外上下文。
</Callout>
